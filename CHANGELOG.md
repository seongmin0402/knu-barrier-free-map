# 변경 이력

## [1.0.0] - 2026-05-09

### 추가됨
- 🗺️ 네이버 지도 API 기반 인터랙티브 맵
- 📍 23개 건물 마커 시스템 (녹색/노란색/빨간색 구분)
- ℹ️ 건물 상세정보 패널
- ♿ 8가지 배리어프리 시설 아이콘
- 🔍 건물명 검색 기능
- 🎚️ 시설별 필터링 기능
- 🎨 고대비 모드 (접근성 향상)
- 📱 반응형 디자인 (모바일/태블릿/데스크톱)
- 👨‍💼 관리자 대시보드 (기본 구조)
- 📖 상세한 문서 (README, 배포 가이드, 기여 가이드)

### 기술적 구현
- Next.js 15 + React 19 + TypeScript
- Tailwind CSS 스타일링
- Custom Hooks (useBuildings, useHighContrast)
- WCAG 2.1 AA 접근성 준수
- 키보드 네비게이션 지원
- ARIA 레이블

### 데이터
- CSV 데이터를 JSON으로 변환
- 23개 건물 정보 포함
- Supabase 이미지 URL 연동
